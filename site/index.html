Quick Start

# Sign up programmatically (no browser needed)
curl -s https://telnyx.com/agent-signup.md

# Install the CLI
npm install -g @telnyx/api-cli

# Authenticate
telnyx auth setup --api-key YOUR_KEY

# Send your first SMS
telnyx message send --from +15551234567 --to +15559876543 --text "Hello"

# Chat with AI
telnyx ai chat "What can Telnyx do?"

# Search for phone numbers
telnyx number search --country US --state CA --json

Capabilities

SMS and MMS

Send and receive messages worldwide. A2P, P2P, toll-free, short codes.

telnyx message send --from +1... --to +1... --text "Hello" POST /v2/messages

Voice and Call Control

Programmable voice with real-time control, conferencing, IVR, recording.

telnyx call dial --from +1... --to +1... POST /v2/calls

AI Voice Assistants

AI-powered assistants with custom personalities, knowledge bases, and tools.

telnyx assistant create --name "Support Bot" POST /v2/ai/assistants

AI Inference

LLM chat completions and embeddings. OpenAI-compatible API.

telnyx ai chat "Explain WebRTC" POST /v2/ai/chat/completions

Phone Numbers

Search, buy, and manage numbers in 140+ countries.

telnyx number search --country US --json GET /v2/available_phone_numbers

Verification (2FA)

Send and verify codes via SMS, voice call, or flash call.

telnyx verify send --phone-number +15551234567 POST /v2/verifications

Fax

Send and receive faxes programmatically.

telnyx fax send --to +1... --media-url https://... POST /v2/faxes

IoT and Wireless

SIM cards, eSIMs, data plans, global connectivity for IoT devices.

telnyx sim list --json GET /v2/sim_cards

Cloud Storage

S3-compatible object storage for media, recordings, and files.

telnyx storage bucket create --name my-bucket PUT /v2/storage/buckets

Private Networking

Private IP mesh networking, cloud interconnects, multi-region infrastructure.

API: POST /v2/networks POST /v2/networks
